              Hey,

              What you need to do is to learn how to write a C extension for Python indeed. I don't know any good documentation about that. Cython can be a good track if it's enough for you – I didn't check it recently and I don't know if the limitations it had has been removed by now, which is likely. Other than that, grabbing the CPython extension module documentation such as https://docs.python.org/2/exte... with some examples stolen from PyPI will probably help more.

              Also not the CFFI module https://cffi.readthedocs.org/e... which can solve that for you more quickly without writing any C. :)
